## 프로젝트 소개
이 프로젝트는 사용자들이 레시피를 찾고, 공유하며, 소통할 수 있는 웹 기반 플랫폼이다. 

주요 기능으로는 회원가입 및 로그인, 레시피 조회, 레시피 작성, 레시피 추천, 레시피 댓글 및 상호작용, 그리고 마이 페이지 관리가 있다. 

레시피 추천시 재료와 조리 과정을 함께 고려한 이중 추천 시스템을 이용하여 레시피 추천의 다양성과 정확성을 제공한다.
